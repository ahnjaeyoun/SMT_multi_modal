# 멀티모달 AI 기반 SMT 불량 탐지·원인 분석 시스템

## 1. 프로젝트 개요

본 프로젝트는 SMT(Surface Mount Technology) 공정에서 발생하는 불량을 조기에 탐지하고,  
불량 원인을 분석하여 현장 작업자가 빠르게 대응할 수 있도록 지원하는  
**멀티모달 AI 기반 불량 탐지·원인 분석 시스템**입니다.

SMT 공정 이미지 데이터와 설비 센서 데이터를 함께 활용하여  
제품의 정상/불량 여부를 판단하고, PR 공정 및 SD 공정의 불량 위치와 유형을 탐지합니다.  
또한 센서 데이터 기반 원인 분석을 통해 불량 발생 가능성이 높은 원인 후보를 제공하고,  
이를 웹 대시보드에서 실시간으로 확인할 수 있도록 구현했습니다.

---

## 2. 개발 배경

기존 SMT 품질 관리 방식은 Reflow 이후 불량이 발견될 경우  
폐기 및 재작업 비용이 크게 증가한다는 문제가 있습니다.

또한 기존 검사 장비의 오탐(False Call)은 검사 병목을 유발하고,  
장비 가동률 저하와 생산성 감소로 이어질 수 있습니다.

본 프로젝트는 이러한 문제를 해결하기 위해  
공정 초기에 AI 기반으로 불량을 탐지하고,  
불량 원인 후보를 함께 제공하는 통합 품질 관리 시스템을 목표로 합니다.

---

## 3. 주요 기능

- 이미지 및 센서 데이터 기반 제품 정상/불량 이진 분류
- PR 공정 불량 위치 및 유형 탐지
- SD 공정 불량 위치 및 유형 탐지
- 센서 데이터 기반 불량 원인 후보 분석
- SHAP 기반 센서별 영향도 시각화
- 실시간 검사 결과 및 신뢰도 확인
- 불량 위치 Bounding Box 시각화
- 공정별 불량 현황 및 추이 확인
- 검수 승인/반려를 통한 품질 이력 관리
- Docker 및 Kafka 기반 실시간 AI 서비스 구성

---

## 4. 데이터 구조

본 프로젝트의 데이터는 이미지 데이터, 라벨링 데이터, 센서 시계열 데이터로 구성됩니다.

### 4.1 이미지 데이터

- SMT 공정 이미지 샘플
- 해상도: 512 x 512
- 정상 / 불량 데이터 포함
- 부품 및 납 상태 확인 가능

### 4.2 Annotation 데이터

- COCO 형식 기반 구조
- Category ID로 결함 유형 정의
- Bounding Box로 결함 위치 표기
- 정상 / 불량 클래스 관리

### 4.3 센서 시계열 데이터

- 온도, 습도, 진동, 가속도, 소음 데이터 수집
- Timestamp 기반 시계열 데이터
- 이미지 단위로 센서 시퀀스 연결
- defect_detected 값을 통해 정상 / 불량 상태 매칭

```text
현장 데이터 수집
    ↓

    ↓
Classification Model

    ↓
